📦 Sistema de Almoxarifado - Centriar

Sistema desenvolvido para gerenciamento de almoxarifado, permitindo o controle de materiais, movimentações de estoque e organização dos recursos utilizados por uma instituição.

O objetivo do sistema é facilitar o controle de entrada e saída de produtos, garantindo maior organização, rastreabilidade e eficiência no gerenciamento de materiais armazenados. Sistemas de almoxarifado permitem registrar movimentações de estoque e acompanhar o saldo de materiais disponíveis, tornando o controle mais eficiente e seguro.

🚀 Funcionalidades

📋 Cadastro de produtos

📦 Controle de estoque

⬆️ Registro de entrada de materiais

⬇️ Registro de saída de materiais

🔎 Consulta de produtos em estoque

📊 Controle e organização do almoxarifado

🧾 Histórico de movimentações

⚙️ Requisitos

Para executar o projeto localmente, você precisa ter instalado:

Servidor Web (Apache recomendado)

PHP 7+

MySQL ou MariaDB

XAMPP / WAMP / Laragon

💻 Instalação

Clone o repositório:

git clone https://github.com/Anderson-Fontes/Sistema-de-Almoxarifado---Centriar.git

Coloque a pasta dentro do diretório do servidor web:

Exemplo (XAMPP):

htdocs/

Crie um banco de dados no MySQL.

Importe o arquivo .sql localizado na pasta do banco de dados.

Configure as credenciais de conexão no arquivo de configuração do banco.

Exemplo:

$host = "localhost";
$user = "root";
$password = "";
$database = "almoxarifado";

Inicie o servidor e acesse no navegador:

http://localhost/Sistema-de-Almoxarifado---Centriar
